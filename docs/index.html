Rocket Translator

Create HTML Components and Use it on Vue, React and Angular

What is Rocket Translator?

Is a HTML translator, which you can create a single file component, and then translate to React, Angular or Vue.

The idea is unify frameworks development, with the same base code.

With Rocket Translator, you going to save some time, creating a HTML component, and share that components beetwen diferent-famework's projects

How To Start?

Is easy! Rocket Translator is in a CLI (Command Line Interface) format.
To start, simply install it on global with your package manager.

#install with NPM
$ npm i -g rocket-translator

#install with Yarn
$ yarn global add rocket-translator

Now let's start to write code!

The basic Rocket Translator functionality is Only HTML.
We can declare a Component State in template

<div>
	<span>Hello {name - state - "World"}!</span>
</div>

Some features in Rocket Translator, works with name-type-value concepts. We declare, name and type, and then we assign a value.

In the previous example, the translator detect {name - state - "World"} and get the name: 'name', type: 'state' and value: 'World'

Then do some works to translate the template on Vue, React or Angular.

<template>
	<div>
		<span>Hello {{name}}!</span>
	</div>
</template>
<script>
	export default {
		name: "MyComponent",
		data() {
			return {
				name: "World"
			}
		}
	}
</script>
import React, {Component} from "react";

class MyComponent extends Component {
	constructor() {
		super();
		this.state = {
			name: "World"
		};
	}
	render() {
		return (
			<div>
				<span>Hello {this.state.name}!</span>
			</div>
		)
	}
}
export default MyComponent;
import {Component} from '@angular/core';

@Component({
	selector: 'my-component-root',
	template:`<div>
		<span>Hola {{name}}</span>
	</div>`
})
export class MyComponent {
	name = "Mundo";	
}
